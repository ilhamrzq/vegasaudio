# Car Audio Installation Booking Web App

A web-based booking application for car audio installation services at VEGAS Audio Variation Shop.
This system allows customers to easily schedule installation services without requiring an account, while providing administrators with a complete dashboard to manage bookings, customers, and reports.

## Features
👤 User Side
* View shop profile and available services
* Browse car audio package categories
* Book audio installation services without login (Input personal data, car type, and audio package)
* Submit feedback, reviews, and suggestions

🔐 Admin Side
* Admin login system
* Manage booking registrations
* Manage customer data
* Handle payment data
* Generate reports
* Manage:
> _**Admin data, Car types, Audio packages, Customer feedback & suggestions**_

## Tech Stack
* Backend: PHP Native
* Frontend: HTML, CSS, JavaScript
* Database: MySQL
